Ques 1. What is encapsulation?
Ans: Encapsulation is defined as the wrapping up of data under a single unit. It is the mechanism that binds together code and the data it manipulates.
This concept is also often used to hide the internal representation, or state of an object from the outside. This is called information hiding.

Ques 2. What are various types of class relationships, explain with example.
Ans : There are three most common relationships among classes in Java that are as follows:

a. Dependence (“Uses-A”): When we create an object of a class inside a method of another class, this relationship is called dependence relationship.
b. Association (“Has-A”): When an object of one class is created as data member inside another class, it is called association relationship in java.
c. Inheritance (“Is-A”): Is-A relationship defines the relationship between two classes in which one class extends another class.
